# Resize events

## Resize values

| Property | default | Description |
| :------- | :------ | :---------- |
| width    | 0       |             |
| height   | 0       |             |
| top      | 0       |             |
| left     | 0       |             |
| right    | 0       |             |
| bottom   | 0       |             |
| x        | 0       |             |
| y        | 0       |             |

## Resize events

| Event name | Argument    | Description                                          |
| :--------- | :---------- | :--------------------------------------------------- |
| onResize   | ResizeState | Called on all resize events. Mainly events specified |
| onMount    | Element     | Called when the component is mounted                 |
| onClean    | null        | Called when the component is removed                 |

## Resize state

```ts
export interface ResizeState<El extends Element = Element> {
        observer: ResizeObserver | null
        listener(): void
        onMount(el: El): void
        onClean(el: null): void
        onResize(self: ResizeState<El>): void
        ref(target: El): void
}
```
