# How to create key event

## create your app

```tsx
import { useKey } from 'rege/react'

export const App = () => {
        const { ref } = useKey(console.log)
        return <span ref={ref} style={style} />
}
```

import { useKey } from '@site/../../packages/gesture/react'

export const App = () => {
        const { ref } = useKey(console.log)
        return <span ref={ref} style={style} />
}

<App />

### with Key component

You can use shorthands Key component

```jsx
import { Key } from 'rege/react'

export const App2 = () => {
        return (
                <Key onKey={console.log}>
                        {({ ref }) => <span ref={ref} style={style} />}
                </Key>
        )
}
```

import { Key } from '@site/../../packages/gesture/react'

export const App2 = () => {
        return (
                <Key onKey={console.log}>
                        {({ ref }) => <span ref={ref} style={style} />}
